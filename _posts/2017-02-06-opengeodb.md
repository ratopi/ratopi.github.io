Published [OpenGeoDB](https://github.com/ratopi/opengeodb) – a git mirror of the official data from [opengeodb.org](http://www.opengeodb.org/).

OpenGeoDB is a public domain database of geographical data for German-speaking countries (Germany, Austria, Switzerland, Belgium, Liechtenstein, Luxembourg), including zip codes, city names, and coordinates.

The original project seems to be fading, so I created a git copy to keep the data accessible. Both CSV and JSON formats are available via GitHub Pages:

- [DE.tab](https://ratopi.github.io/opengeodb/DE.tab) / [DE.tab.json](https://ratopi.github.io/opengeodb/DE.tab.json) – Germany
- [AT.tab](https://ratopi.github.io/opengeodb/AT.tab) / [AT.tab.json](https://ratopi.github.io/opengeodb/AT.tab.json) – Austria
- [CH.tab](https://ratopi.github.io/opengeodb/CH.tab) / [CH.tab.json](https://ratopi.github.io/opengeodb/CH.tab.json) – Switzerland
- [PLZ.tab](https://ratopi.github.io/opengeodb/PLZ.tab) – Zip codes

The data is public domain.
