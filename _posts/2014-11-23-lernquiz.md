Published [LernQuiz](/projects/lernquiz) – a small web application that presents multiple-choice questions about German federal states (capitals, coats of arms, flags, maps), European capitals, and national flags.

Written in Elm, running completely serverless. No external fonts, no CSS frameworks, no JavaScript libraries – everything runs locally in the browser.

Try it out: [ratopi.github.io/lernquiz](https://ratopi.github.io/lernquiz/)
