Published a small Erlang library for parsing extended [M3U](https://de.wikipedia.org/wiki/M3U) playlist files.

Currently provides parsers for the extended M3U format. Support for [PLS](https://de.wikipedia.org/wiki/PLS_(Dateiformat)) and [HLS (RFC 8216)](https://www.rfc-editor.org/rfc/rfc8216) is planned.

Source code on [GitHub](https://github.com/ratopi/playlist).
