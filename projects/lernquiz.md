A small web application that presents multiple-choice questions about German federal states (capitals, coats of arms, flags, maps), European capitals, and national flags.

Written in [Elm](https://elm-lang.org/), running completely serverless as a static website. No external fonts, CSS frameworks, or JavaScript libraries – everything runs locally.

### Try it

👉 [ratopi.github.io/lernquiz](https://ratopi.github.io/lernquiz/)

### Adding quizzes

Quiz definitions are JSON files in `src/quiz/`. Each quiz has a category, title, description, and a list of questions with optional images.

Source code on [GitHub](https://github.com/ratopi/lernquiz).
