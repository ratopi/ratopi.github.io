A tool that downloads and converts the official German charging station register (*Ladesäulenregister*) from the Bundesnetzagentur into a more usable JSON format.

Written in Erlang. The converted data is updated weekly via GitHub Actions and publicly available as a JSON file.

### How it works

The tool scrapes the current CSV download URL from the [Bundesnetzagentur E-Mobility page](https://www.bundesnetzagentur.de/DE/Fachthemen/ElektrizitaetundGas/E-Mobilitaet/start.html), streams the CSV via HTTP, parses it on the fly using a continuation-based CSV parser, and converts it into a nested JSON structure – including address, geo coordinates, charging points with plug types and power ratings, payment methods, and opening hours.

An update check via HTTP HEAD avoids unnecessary downloads.

### JSON access

The current `ladesaeulen.json` is available at:

**[https://ratopi.github.io/ladesaeule/ladesaeulen.json](https://ratopi.github.io/ladesaeule/ladesaeulen.json)**

### Note

The CSV file is obviously generated by hand using Excel. The format may change without notice. If you spot any issues, please open an [issue on GitHub](https://github.com/ratopi/ladesaeule/issues).

Source code on [GitHub](https://github.com/ratopi/ladesaeule).
